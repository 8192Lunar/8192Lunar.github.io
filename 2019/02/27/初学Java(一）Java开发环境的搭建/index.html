学习Java，如果用户采用记事本、UltraEdit等文字处理软件书写Java源文件，则需下载JDK来编译执行Java源文件。
具体步骤：

  1. 下载JDK开发工具包JAVA SE（标准版，开发部署在桌面及服务器上的应用程序）

  2. 注意要同意许可协议。

  3. 下载完成后双击启动安装程序，一路直接下一步，但要注意选择并记住安装位置。注：每个人的安装位置可不同。

  4. 接下来是设置环境变量Path(以win10和JDK1.8为例)
  5. 右键打开此电脑的属性，选择左侧控制面板页面的的高级系统设置，点击右下角的环境变量，找到下方系统变量的Path，点击编辑，在编辑环境变量页面选择新建，输入JDK所在路经的bin文件夹，如图（小编将JDK放在了C盘的C:\Program Files下）

  6. JDK1.8的环境配置还没结束，我们还有再新建一个环境变量Classpath并设置其值：（别忘了最上面的【.】）， 在新建时输入【.;C:\Program Files\Java\jdk1.8.0_05\lib\tools.jar;E:\Program Files\Java\jdk1.6.0_05\jre\li\dt.jar】
    【注意开头是句号加分号】（“;”分号主要用于分隔不同文件）

  7. 最后是验证验证JDK是否安装好：
    在命令窗口输入java -version

  8. 如果出现版本号，就说明环境配置成功，反之，则说明配置失败，要重新检查每一步配置步骤是否有误。

以上就是Java开发环境的配置过程，如有错漏，请私信小编指出，小编也是初学者，望能与诸位共勉。
